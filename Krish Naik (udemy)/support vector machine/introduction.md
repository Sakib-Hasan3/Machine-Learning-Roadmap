# 3) рж╕рж┐ржжрзНржзрж╛ржирзНржд рж╕рзАржорж╛рж░рзЗржЦрж╛ ржУ ржорж╛рж░рзНржЬрж┐ржи ржмрзЗрж░ ржХрж░рж╛

plt.scatter(X[y==1, 0], X[y==1, 1], c='tomato', label='ржХрзНрж▓рж╛рж╕ 1', edgecolor='k')

# рж╕рж┐ржжрзНржзрж╛ржирзНржд рж╕рзАржорж╛рж░рзЗржЦрж╛ ржУ ржорж╛рж░рзНржЬрж┐ржи

plt.title('Linear SVM: рж╕рзАржорж╛рж░рзЗржЦрж╛, ржорж╛рж░рзНржЬрж┐ржи ржУ Support Vectors')
