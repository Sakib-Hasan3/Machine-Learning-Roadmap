# тнР Python Example (Ordinal Logistic Regression тАФ statsmodels)

```python
import pandas as pd
import statsmodels.api as sm
from statsmodels.miscmodels.ordinal_model import OrderedModel

# Sample data
df = pd.DataFrame({
    'study_hours': [2, 5, 7, 1, 4, 6],
    'attendance': [60, 80, 90, 40, 70, 85],
    'grade': ['C', 'B', 'A', 'C', 'B', 'A']
})

# Convert to ordered categorical
df['grade'] = pd.Categorical(df['grade'], categories=['C', 'B', 'A'], ordered=True)

model = OrderedModel(df['grade'], df[['study_hours', 'attendance']],
                     distr='logit')

result = model.fit(method='bfgs')
print(result.summary())
```
